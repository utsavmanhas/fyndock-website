About this mock-up. This is a redesign of the Services section as it could appear in a future iteration of the home page. Five changes vs. the current version: (1) added a "Best for" persona line above each card so visitors can self-select faster; (2) replaced "Starting price" with "From ₹X" — feels less hedged; (3) added a small sample-preview thumb strip inside each card so the deliverable is visible without scrolling to the Deliverables Preview section; (4) introduced a comparison strip below the cards showing what's common across tiers (CA review, 24–48h onboarding, no lock-in); (5) added a top-right toggle (Monthly / Quarterly / Annual) as visual scaffolding for future annual-billing pricing. All design tokens, typography, and hover behaviour match the existing system. Use this file as a Vercel preview deployment to A/B against the current services grid before deciding whether to roll it in.
